# About MountBlue Prep

MountBlue Prep is a structured coding challenge designed to help aspiring software engineers build strong problem-solving abilities and become industry-ready.

The challenge consists of solving multiple coding problems across various DSA topics using programming languages like Python, Java, C++, or JavaScript.

For every problem, I try to follow this approach:

1. Understand the problem statement
2. Identify input/output constraints
3. Write a brute force solution
4. Optimize the solution
5. Analyze time and space complexity
6. Write clean and maintainable code
